# Arturo Portfolio

Minimal Next.js portfolio ready for GitHub Pages.

## Update content

Edit `src/data/profile.ts`.

## Replace photo

Replace `public/profile-photo.svg` with your own image and keep the same file name, or update the path in `src/data/profile.ts`.

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Push to `main`, then enable GitHub Pages and choose `GitHub Actions` as the source.
